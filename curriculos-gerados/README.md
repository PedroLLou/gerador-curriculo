# 📄 Currículos Gerados / Generated Resumes

Uma **subpasta por vaga** / one **subfolder per job**:

```
<empresa>-<cargo>-AAAA-MM-DD/        (minúsculo, com hífens / lowercase, hyphens)
├── vaga.md            # análise da vaga / job analysis
├── scorecard.md       # aderência / fit scorecard
├── curriculo.md       # fonte / source
├── curriculo.html     # versão estilizada / styled version
├── curriculo.pdf      # versão humana / human version
└── curriculo-ats.docx # versão ATS / ATS version
```

## Como gerar / How to generate
Envie a descrição da vaga ao assistente (que segue o `CLAUDE.md`).
Send the job description to the assistant (which follows `CLAUDE.md`).
